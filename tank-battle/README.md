# 坦克大战

一个使用HTML5 Canvas开发的经典坦克大战游戏。

## 🎮 游戏特性

- **玩家控制**：使用方向键移动绿色坦克
- **射击系统**：按空格键发射炮弹
- **敌机生成**：自动生成红色敌方坦克
- **计分系统**：击毁敌机获得分数
- **视觉效果**：爆炸粒子效果
- **响应式设计**：支持不同屏幕尺寸

## 🕹️ 操作说明

- **↑ ↓ ← →**：移动坦克
- **空格键**：发射炮弹
- **开始游戏**：点击"开始游戏"按钮
- **重新开始**：点击"重新开始"按钮

## 🚀 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 🎯 游戏规则

1. 玩家控制绿色坦克在战场移动
2. 红色敌方坦克从顶部不断生成并向下移动
3. 玩家可以射击击毁敌方坦克获得分数
4. 如果被敌方坦克撞击或被敌方炮弹击中，游戏结束
5. 随着得分增加，游戏难度会逐渐提升
